<h2><a href="https://www.geeksforgeeks.org/problems/sum-of-digit-is-pallindrome-or-not2751/1">Palindrome Digit Sum</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a number&nbsp;<strong>n</strong>. Return <strong>true </strong>if the digit sum(or sum of digits) of n is a Palindrome number otherwise <strong>false</strong>.<br></span><span style="font-size: 18px;">A Palindrome number is a number that stays the same when reversed</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 56
<strong>Output: </strong>true
<strong>Explanation: </strong>The digit sum of 56 is 5+6 = 11. Since, 11 is a palindrome number.Thus, answer is true.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 98
<strong>Output: </strong>false
<strong>Explanation: </strong>The digit sum of 98 is 9+8 = 17. Since 17 is not a palindrome,thus, answer is false.</span></pre>
